#### 4. 반복문으로 네모 출력

두개의 정수 n 과 m 이 주어졌을 때 , 가로의 길이가 n, 세로의 길이가 m 인 직사각형 형태를
별 (*) 문자를 이용하여 출력하시오 . 단 , 반복문을 사용하여 작성하시오

```python
n = int(input())
m = int(input())

for i in range(m):
    for j in range(n):
        print('*', end='')

    print('*')
```

#### 5. 조건 표현식

주어진 코드의 조건문을 조건 표현식으로 바꾸어 작성하시오

```python
temp = 36.5
if temp >= 37.5:
    print('입실 불가')
else:
    print('입실 가능')

#조건 표현식 true if expression else false
result = '입실불가' if temp >= 37.5 else '입실 가능'
print(result)
```
